Практическая работа №3

Установка iSCSI Target на Ubuntu 22.04 LTS

Задание:

1) Начать документировать процесс выполнения

2) Установить и настроить iSCSI Target из пакета istgt на машине для второго практического занятия, создать и экспортировать 2 LUN по 512 МБ с доступом с паролем для пользователя из лабораторной работы №1
Для определения размещения устройств разделить номер бригады на 2 и найти остаток от деления:
		0 – RAM (в оперативной памяти, tmpfs)
		1 – block (на блочном устройстве, LVM)
		2 – file (в файле)
	При нехватке ресурсов  – запросить увеличение у преподавателя (память:  cat /proc/memory, блочные устройства: lslbk, место на дисках: df  -h)

3) Настроить firewall для удаленного доступа к iSCSI Target на данной машине по сети.

4) Проверить работоспособность iSCSI Target из п.3, в том числе после перезагрузки машины.

5) Завершить процесс документирования, дополнить документацию необходимыми комментариями и подробным описанием всех встреченных в процессе сложностей. Допустимые форматы отчета: docx, pdf. Индивидуальный отчет после защиты загрузить в DiSpace3 (1 экземпляр от бригады).
