                Лабораторная работа #1

Тема: Базовые операции в ОС GNU/Linux на базе дистрибутива CentOS 6.10

Задание:
1) В каталоге /tmp создать подкаталог, совпадающий с именем пользователя.
   Проверить, что созданный каталог принадлежит пользователяю (stat).
   Запретить "прочим" просматривать каталог (chmod)

2) Написать программу на языке bash, выполняющую следующие действия:
     a) Программа создает файл отчета в каталоге /tmp, имя файла создается
        по формуле: "имя пользователя, запустившего программу" + "текущая дата и
        время" + ".txt"
     b) Рекурсивно проверяет время модификации файлов в каталоге, указанном
        в параметрах вызова программы, находит все изменившиеся файлы с момента
        последней проверки и включает их полные имена в файл отчета. Для примера
        использовать каталог из п.1 текущего задания
     c) Если файл отчета не пуст, то отправляет файл отчета по почте
        локальному пользователю, от имени которого была запущена программа
     d) Сохраняет дату последней проверки в домашнем каталоге пользователя
     e) Удаляет временный файл отчета
        
3) Добавить программу из п.2 в запуск по расписанию (crontab -e), интервал
   запуска - раз в 5 минут.

4) Файл программы из п.2, результат работы вызова программы crontab -l и
   одно из писем (с полным заголовком), созданных в результате успешного
   выполенния п.2.с поместить в файл-образ .iso (mkisofs), полученный файл
   скачать с сервера при помощи программы pscp.exe

Содержание отчета:
